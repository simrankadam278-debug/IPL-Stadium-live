# 🏏 IPL Cricket Hub

A modern and interactive IPL Cricket web application built using React, Vite, and Tailwind CSS.  
The project provides live-style match interfaces, IPL team and player information, stadium maps, rankings, statistics dashboards, and a premium sports-themed UI experience.

---

## 🚀 Features

### 🎯 Core Features
- Live IPL Match Score Cards
- IPL Points Table & Rankings
- Team & Player Profiles
- Match Schedules & Fixtures
- Stadium Details & Interactive Maps
- Cricket News & Highlights
- Search Functionality
- Responsive Modern UI

---

## 🌟 Special Features

### 🗺️ Stadium Maps
- “Open Stadium Map” button
- Interactive maps using React Leaflet / Google Maps API
- Stadium markers and location details

### 🌙 Dark & Light Theme
- Fully functional theme toggle
- Saves theme using localStorage
- Smooth transition animations
- Dynamic icons for dark/light mode

### 📊 Cricket Statistics Dashboard
- Animated charts for:
  - Team performance
  - Player stats
  - Match analysis

### ⚡ Premium UI/UX
- Glassmorphism cards
- Gradient sporty design
- Hover animations
- Smooth page transitions
